BU_VOTEBOTS_DESCRIPTION = Initiate a vote to enable or disable bots. Requires majority vote to pass.
BU_VOTEBOTS_DESCRIPTION_AL = Alias of !votebots
BU_VOTEBOTS_IN_PROGRESS = A vote to enable/disable bots is already in progress. Please wait for it to conclude before starting a new one.
BU_VOTEBOTS_DISABLED = Bots have been disabled by vote.
BU_VOTEBOTS_ENABLED = Bots have been enabled by vote.
BU_VOTEBOTS_TIED = The vote to enable/disable bots has ended in a tie. No changes have been made.
BU_VOTEBOTS_TITLE_ENABLE = Want to play with bots? (Bots are currently disabled)
BU_VOTEBOTS_TITLE_DISABLE = Want to play with bots? (Bots are currently enabled)
BU_VOTEBOTS_ENABLE = Yes, enable bots
BU_VOTEBOTS_DISABLE = No, disable bots
BU_VOTEBOTS_VOTE_DISABLED = This vote is disabled
BU_VOTEBOTS_VOTE_NO_EFFECTS = There are no bots to manage
BU_QUOTA_DETECT_NOTICE = There are currently %d bots on the server. You can disable bots by typing !votebots in chat.
