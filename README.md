# AMXX Miscellaneous Plugins

A collection of small, experimental, and miscellaneous AMX Mod X plugins.

This repository exists as a central place for plugins that:

- May be too small (feature or otherwise) to justify their own repository
- May be experimental or created for testing purposes
- May be incomplete, unstable, or abandoned
- Do not require long-term maintenance
- May be partially or heavily generated using LLM tools
- May be heavily hardcoded

These plugins are provided as is, and no support will be given.
